## SEO ומדיה חברתית

- עדכנו את `title`, תגית `meta description` ואת `og:title`, `og:description`, `og:url`, `og:image` בהתאם לדומיין הסופי.
- עדכנו ב-JSON-LD את `name`, `url`, `telephone` ואת `sameAs`.
